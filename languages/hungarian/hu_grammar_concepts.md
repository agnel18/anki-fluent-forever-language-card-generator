# Hungarian Grammar Concepts

## Overview

Hungarian (magyar) is a Uralic language in the Finno-Ugric branch. It is NOT Indo-European. It uses the Latin alphabet with 14 additional characters: á, é, í, ó, ö, ő, ú, ü, ű (vowels) and cs, dz, dzs, gy, ly, ny, sz, ty, zs (digraphs/trigraphs counted as single letters).

## Key Features

### 1. Agglutinative Morphology
Hungarian builds words by adding suffixes to stems. A single word can encode subject, object, tense, mood, and possession:
- **olvas** (read) → **olvas-om** (I read it [definite]) → **olvas-hat-om** (I can read it) → **el-olvas-hat-om** (I can read it through)

### 2. Vowel Harmony
Suffixes change form based on whether the stem contains front or back vowels:
- **Back vowels:** a, á, o, ó, u, ú
- **Front vowels:** e, é, i, í, ö, ő, ü, ű
- Examples: **ház-ban** (in the house) vs **kert-ben** (in the garden); **asztal-nak** (for the table) vs **szék-nek** (for the chair)

### 3. 18 Grammatical Cases
Hungarian has one of the richest case systems among living languages:

| Case | Suffix | Example | Meaning |
|------|--------|---------|---------|
| Nominative | -∅ | ház | house |
| Accusative | -t/-at/-ot/-et/-öt | házat | house (obj) |
| Dative | -nak/-nek | háznak | for the house |
| Instrumental | -val/-vel | házzal | with the house |
| Causal-final | -ért | házért | for (the sake of) the house |
| Translative | -vá/-vé | házzá | into (becoming) a house |
| Inessive | -ban/-ben | házban | in the house |
| Superessive | -n/-on/-en/-ön | házon | on the house |
| Adessive | -nál/-nél | háznál | at the house |
| Sublative | -ra/-re | házra | onto the house |
| Delative | -ról/-ről | házról | from (off) the house |
| Elative | -ból/-ből | házból | out of the house |
| Illative | -ba/-be | házba | into the house |
| Allative | -hoz/-hez/-höz | házhoz | to the house |
| Ablative | -tól/-től | háztól | away from the house |
| Terminative | -ig | házig | until/as far as the house |
| Essive-formal | -ként | házként | as a house |
| Distributive | -nként | házonként | per house |

### 4. Definite vs Indefinite Conjugation
Hungarian verbs have two complete conjugation paradigms:
- **Indefinite (alanyi):** used when the object is indefinite or absent
  - Olvasok egy könyvet. (I read a book.)
- **Definite (tárgyas):** used when the object is definite (has "a/az", possessive, proper noun)
  - Olvasom a könyvet. (I read the book.)

### 5. Preverbs (Igekötők)
Verbal prefixes that modify verb meaning, similar to German separable prefixes:
- **meg-** (completion): ír → megír (write → finish writing)
- **el-** (away): megy → elmegy (goes → goes away)
- **ki-** (out): jön → kijön (comes → comes out)
- **be-** (in): megy → bemegy (goes → goes in)
- **fel-** (up): áll → feláll (stands → stands up)
- **le-** (down): ül → leül (sits → sits down)
- **vissza-** (back): jön → visszajön (comes → comes back)
- **össze-** (together): gyűjt → összegyűjt (collects → gathers together)

### 6. Postpositions
Hungarian uses postpositions instead of prepositions:
- **mellett** (beside): a ház mellett (beside the house)
- **alatt** (under): az asztal alatt (under the table)
- **felett** (above): a fej felett (above the head)
- **mögött** (behind): az auto mögött (behind the car)
- **előtt** (in front of): az iskola előtt (in front of the school)
- **között** (between): a házak között (between the houses)

### 7. Possessive Suffixes
Possession is expressed with suffixes on the possessed noun:
- **-m** (my): házam (my house)
- **-d** (your): házad (your house)
- **-ja/-je** (his/her): háza (his/her house)
- **-nk** (our): házunk (our house)
- **-tok/-tek/-tök** (your pl.): házatok (your house)
- **-juk/-jük** (their): házuk (their house)

### 8. Word Order — Topic-Focus
Default is SVO, but word order is flexible. The **focus position** (immediately before the verb) receives emphasis:
- **Péter** olvas. (Péter reads.) — neutral
- **A könyvet** olvassa Péter. (It's THE BOOK that Péter reads.) — focus on object

## Complexity Levels

### Beginner
- Basic SVO sentences
- Nominative, accusative, dative cases
- Indefinite conjugation (present tense)
- Common postpositions (ban/ben, hoz/hez/höz)
- Personal pronouns with basic case marking
- Simple preverbs (meg-, el-)

### Intermediate
- Definite vs indefinite conjugation distinction
- 10+ case endings with vowel harmony
- Past tense conjugation
- Possessive suffixes (1st/2nd/3rd person singular)
- Conditional mood
- Preverb separation in questions/negation
- Compound words

### Advanced
- All 18 cases
- Full possessive paradigm (all persons, singular/plural)
- Causative verbs (-tat/-tet)
- Potential verbs (-hat/-het)
- Subjunctive mood
- Complex preverb behavior (word order changes)
- Focus-topic structure analysis
- Vowel harmony exceptions (mixed-vowel words)

## Example Sentence Breakdowns

### Beginner: "A fiú almát eszik."
(The boy eats an apple.)
- **A** — definite article
- **fiú** — boy [noun, nominative]
- **almát** — apple + accusative -t [noun, accusative case]
- **eszik** — eats [verb, 3rd person singular, indefinite conjugation, present]

### Intermediate: "Megírtam a levelet tegnap."
(I finished writing the letter yesterday.)
- **Meg-** — preverb (completion) [preverb]
- **ír-t-am** — write + past -t + 1st person definite -am [verb, definite conjugation, past]
- **a** — the [definite article]
- **level-et** — letter + accusative -t [noun, accusative]
- **tegnap** — yesterday [adverb]

### Advanced: "A barátomnak adnám a könyvemet, ha kérné."
(I would give my book to my friend, if he/she asked for it.)
- **A** — the [definite article]
- **barát-om-nak** — friend + possessive 1sg -om + dative -nak [noun, possessive, dative]
- **ad-ná-m** — give + conditional -ná + 1sg definite -m [verb, definite conditional]
- **a** — the [definite article]
- **könyv-em-et** — book + possessive 1sg -em + accusative -et [noun, possessive, accusative]
- **ha** — if [conjunction]
- **kér-né** — ask + conditional 3sg -né [verb, conditional 3sg]
