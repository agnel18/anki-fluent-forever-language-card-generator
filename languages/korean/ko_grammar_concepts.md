# Korean Grammar Concepts

## Overview
Korean (한국어, Hangugeo) is a Koreanic language spoken by ~80 million people.
It uses the Hangul (한글) writing system — an alphabetic script organized in syllable blocks.

## Script & Writing System
- **Hangul syllable blocks**: Each block = initial consonant + vowel + optional final consonant
- **Example**: 한 = ㅎ (h) + ㅏ (a) + ㄴ (n)
- **Spaces between words**: Unlike Japanese, Korean uses spaces between words
- **Text direction**: Left-to-right, horizontal

## Word Order
- **SOV**: Subject – Object – Verb (verb always at the end)
- **Flexible**: Subject and object can be dropped or reordered with particles
- **Head-final**: Modifiers precede the head (adjective before noun, relative clause before noun)

## Key Grammatical Features

### 1. Particles (조사, josa)
Postpositional markers attached directly to nouns:
| Particle | Function | Example |
|----------|----------|---------|
| 은/는 | Topic marker | 나**는** 학생이다 (As for me, I am a student) |
| 이/가 | Subject marker | 비**가** 온다 (Rain comes / It's raining) |
| 을/를 | Object marker | 밥**을** 먹다 (to eat rice) |
| 에 | Location/time | 학교**에** 가다 (to go to school) |
| 에서 | Location of action | 집**에서** 공부하다 (to study at home) |
| 의 | Possessive | 나**의** 책 (my book) |
| 도 | Also/too | 나**도** (me too) |
| 와/과, 하고, (이)랑 | And/with | 사과**와** 배 (apple and pear) |
| 로/으로 | Direction/means | 버스**로** 가다 (go by bus) |

### 2. Verb Conjugation & Speech Levels
Korean verbs conjugate for tense, aspect, mood, and politeness:
- **Dictionary form**: 먹다 (to eat)
- **Polite (해요체)**: 먹어요
- **Formal (합쇼체)**: 먹습니다
- **Casual (해체)**: 먹어
- **Past tense**: 먹었어요
- **Future/intention**: 먹을 거예요
- **Progressive**: 먹고 있다

### 3. Honorific System (높임말)
- **Subject honorific**: -(으)시- suffix (선생님이 오**시**다 = The teacher comes)
- **Humble forms**: Special vocabulary (먹다 → 드시다 [honorific], 먹다 → 들다 [humble])
- **Speech levels**: 7 levels, commonly used 4: 합쇼체 (formal polite), 해요체 (informal polite), 해체 (casual), 해라체 (plain)

### 4. Connective Endings (연결 어미)
- **-고**: and (sequential/listing): 먹고 마시다 (eat and drink)
- **-서/아서/어서**: because/and then: 비가 와서 못 갔어요 (Because it rained, I couldn't go)
- **-(으)면**: if/when: 시간이 있으면 (if there's time)
- **-지만**: but/although: 비싸지만 좋아요 (It's expensive but good)

### 5. Adjectives (형용사)
Korean adjectives conjugate like verbs:
- **Dictionary form**: 크다 (to be big)
- **Modifying form**: 큰 (big [before noun])
- **Polite**: 커요 (it's big)

### 6. Counters/Classifiers (단위 명사)
Used with numbers: 사과 세 **개** (three apples), 사람 두 **명** (two people)

## Complexity Levels

### Beginner
- Basic particles: 은/는, 이/가, 을/를
- Simple tenses: present, past
- Polite speech level (해요체)
- Core word classes: noun, verb, adjective, adverb

### Intermediate
- More particles: 에, 에서, 도, 와/과
- Connective endings: -고, -서, -면
- Formal speech level (합쇼체)
- Verb modifiers, counters

### Advanced
- Honorific suffixes: -(으)시-
- Complex connectives: -지만, -는데, -(으)ㄹ 때
- Passive/causative constructions
- Nominalizers: -기, -(으)ㅁ, -는 것
- Quotative: -고 하다 (indirect speech)

## Example Sentences with Breakdowns

### 1. 저는 학생입니다. (I am a student.)
| Word | Role | Explanation |
|------|------|-------------|
| 저 | pronoun | I (humble form) |
| 는 | topic_marker | topic marking particle |
| 학생 | noun | student |
| 입니다 | copula | is/am (formal polite) |

### 2. 오늘 날씨가 좋아요. (The weather is nice today.)
| Word | Role | Explanation |
|------|------|-------------|
| 오늘 | adverb | today |
| 날씨 | noun | weather |
| 가 | subject_marker | subject marking particle |
| 좋아요 | adjective | good/nice (polite form) |

### 3. 어머니께서 음식을 만드셨어요. (Mother made food. [honorific])
| Word | Role | Explanation |
|------|------|-------------|
| 어머니 | noun | mother |
| 께서 | honorific_particle | honorific subject marker |
| 음식 | noun | food |
| 을 | object_marker | object marking particle |
| 만드셨어요 | honorific_verb | made (honorific + past + polite) |
