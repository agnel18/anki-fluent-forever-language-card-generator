# Language Learning Template.apkg

## What is this file?

This is a **pre-configured Anki deck template** that includes:

- ✅ Deck + note type with 8 fields:
  - File Name
  - What is the Word?
  - Meaning of the Word
  - Arabic Sentence (field name kept for compatibility)
  - IPA Transliteration
  - English Sentence
  - Sound
  - Image
- ✅ Optimized card templates (front/back styling)
- ✅ Supports both RTL and LTR languages

## How to use it

1. Open Anki
2. Click **File** → **Import**
3. Select `Language Learning Template.apkg`
4. Click **Open**
5. ✅ Done!

## Creating your own template

If `Language Learning Template.apkg` is not included in this folder, you need to create it:

1. Follow the instructions in [ANKI_SETUP.md](../ANKI_SETUP.md) (Manual Setup section)
2. Export the deck as `.apkg`
3. Save it here as `Language Learning Template.apkg`

## For other languages

To adapt this template for any language:

1. Import this `.apkg` file
2. Rename the deck to your target language
3. (Optional) Rename the note type if you prefer
4. Export the deck again with your new language name

No other changes needed!
