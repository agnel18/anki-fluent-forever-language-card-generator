# Arabic Template.apkg

## What is this file?

This is a **pre-configured Anki deck template** that includes:

- ✅ **Arabic** deck (ready to use)
- ✅ **Arabic Card** note type with 8 fields:
  - File Name
  - What is the Word?
  - Meaning of the Word
  - Arabic Sentence
  - IPA Transliteration
  - English Sentence
  - Sound
  - Image
- ✅ Optimized card templates (front/back styling)
- ✅ Right-to-left text support for Arabic

## How to use it

1. Open Anki
2. Click **File** → **Import**
3. Select `Arabic Template.apkg`
4. Click **Open**
5. ✅ Done!

## Creating your own template

If `Arabic Template.apkg` is not included in this folder, you need to create it:

1. Follow the instructions in [ANKI_SETUP.md](../ANKI_SETUP.md) (Manual Setup section)
2. Export the deck as `.apkg`
3. Save it here as `Arabic Template.apkg`

## For other languages

To adapt this template for Spanish, French, Japanese, etc.:

1. Import this `.apkg` file
2. In Anki, click **Tools** → **Manage Note Types**
3. Select **Arabic Card** → **Rename** → change to your language
4. Export the deck again with your new language name

No other changes needed!
